欢迎来到WeAuth使用手册！
===================
欢迎您使用WeAuth，这是一个轻量化的web服务，用来连接微信公众号与您的Minecraft服务器。

特别感谢 `Fallen_Breath <https://github.com/Fallen-Breath>`__ 开发的 `MCDReforged <https://github.com/MCDReforged/MCDReforged>`__ 带给我们启发，并展示出Python社区无穷的活力。

WeAuth在 `GNU GENERAL PUBLIC LICENSE Version 3 (GPLv3) <https://github.com/TomatoCraftMC/WeAuth/blob/main/LICENSE>`__ 协议下开源，因此WeAuth是一款自由软件，您可以自由下载并部署到您的MC服务系统当中，无论是收费服务器或者是公益服。

如果您开发的应用需要WeAuth作为依赖，或者您修改并重新发布了WeAuth，请遵守GPLv3协议开源。
